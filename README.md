# anderlfrias.dev

Personal portfolio and professional website of **Anderson Frias**.

This project will serve as a central place to showcase my work, technical projects, and professional background as a software developer.

The goal is to create a fast, SEO-friendly portfolio with a modern interactive design.

## Tech Stack

Planned technologies for this project:

* **Next.js**
* **TypeScript**
* **Tailwind CSS**
* **Framer Motion**
* **next-intl** (for internationalization)

## Features (planned)

* Personal portfolio
* Project showcase
* Multi-language support (Spanish / English)
* Interactive hero section
* SEO optimized pages

## Status

🚧 This project is currently under development.

Design, animations, and content are being built progressively.
